Chapter 3: JavaScript Variables

What are Variables?

In JavaScript, variables are used to store data values. JavaScript is a dynamically typed language, which means data types are determined at runtime.

To declare a variable in JavaScript, we use the keywords var, let, and const.

1. var Keyword

The var keyword is the older way of declaring variables in JavaScript. It is function-scoped and accessible throughout the function where it is declared.

Example:

var age = 20;
console.log(age);   // Output: 20

age = 22;
console.log(age);   // Output: 22

A variable declared with var can be reassigned and redeclared within the same scope.

2. let Keyword

The let keyword was introduced to overcome issues associated with var. It is block-scoped, meaning it is accessible only within the block where it is declared.

Example:

let name = "Abhishek";
console.log(name);   // Output: Abhishek

name = "Rahul";
console.log(name);   // Output: Rahul

Block Scope Example:

if (true) {
  let x = 20;
  console.log(x);   // Output: 20
}

The variable x cannot be accessed outside the block.

3. const Keyword

The const keyword is used to declare constant variables. Once a value is assigned, it cannot be reassigned. It is also block-scoped like let.

Example:

const PI = 3.14;
console.log(PI);   // Output: 3.14

Reassigning a const variable is not allowed and will result in an error.

Reassignment of Variables

In JavaScript, a variable can be reassigned by using the assignment operator (=). 

let value = 10;
value = 25;
console.log(value);   // Output: 25

When reassigning a variable, the keyword is not used again.

Summary

  • var – function scoped, can be redeclared
  • let – block scoped, preferred for variables
  • const – block scoped, value cannot be changed
