Chapter 21: MongoDB CRUD Operations

What is CRUD?

CRUD stands for Create, Read, Update, Delete. These are the four basic operations performed on database data.

1. Create Operation (Insert)

The create operation is used to insert documents into a collection.

Insert One Document

db.students.insertOne({
  name: "Amit",
  age: 21,
  course: "MERN"
});

Insert Multiple Documents

db.students.insertMany([
  { name: "Neha", age: 22, course: "MERN" },
  { name: "Rahul", age: 20, course: "MEAN" }
]);

2. Read Operation (Find)

The read operation is used to retrieve data from a collection.

Find All Documents

db.students.find();

Find with Condition

db.students.find({ age: 21 });

Find One Document

db.students.findOne({ name: "Amit" });

3. Update Operation

The update operation is used to modify existing documents.

Update One Document

db.students.updateOne(
  { name: "Amit" },
  { $set: { age: 22 } }
);

Update Multiple Documents

db.students.updateMany(
  { course: "MERN" },
  { $set: { status: "active" } }
);

4. Delete Operation

The delete operation is used to remove documents from a collection.

Delete One Document

db.students.deleteOne({ name: "Rahul" });

Delete Multiple Documents

db.students.deleteMany({ course: "MEAN" });

Important MongoDB Operators

  • $set – Update specific fields
  • $gt – Greater than
  • $lt – Less than
  • $in – Match any value in array

Example Using Operators

db.students.find({ age: { $gt: 20 } });

Summary

  • CRUD operations manage database data
  • MongoDB uses insert, find, update, delete
  • Operators help filter and update data
