Chapter 29: API Integration in React (Fetch / Axios)

What is API Integration?

API integration means connecting a React frontend with a backend server (Node.js / Express) to send and receive data using HTTP requests.

Why API Integration is Important?

  • Fetch data from database
  • Submit form data to server
  • Build dynamic applications
  • Connect frontend with backend

Common HTTP Methods

  • GET – Fetch data
  • POST – Send data
  • PUT – Update data
  • DELETE – Remove data

Using Fetch API

Fetch API is a built-in JavaScript method used to make HTTP requests.

GET Request Example

useEffect(() => {
  fetch("http://localhost:5000/api/users")
    .then(res => res.json())
    .then(data => setUsers(data));
}, []);

POST Request Example

fetch("http://localhost:5000/api/users", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    name: "Amit",
    email: "amit@gmail.com"
  })
});

Handling Errors in Fetch

fetch(url)
  .then(res => {
    if (!res.ok) {
      throw new Error("Network error");
    }
    return res.json();
  })
  .catch(err => console.log(err));

Using Axios

Axios is a popular third-party library used for making HTTP requests. It is easier and cleaner than Fetch.

Installing Axios

npm install axios

GET Request using Axios

import axios from "axios";

useEffect(() => {
  axios.get("http://localhost:5000/api/users")
    .then(res => setUsers(res.data));
}, []);

POST Request using Axios

axios.post("http://localhost:5000/api/users", {
  name: "Neha",
  email: "neha@gmail.com"
});

PUT Request using Axios

axios.put("http://localhost:5000/api/users/1", {
  name: "Updated Name"
});

DELETE Request using Axios

axios.delete("http://localhost:5000/api/users/1");

Using JWT Token with Axios

axios.get("/dashboard", {
  headers: {
    Authorization: token
  }
});

Handling Loading & Error State

const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);

useEffect(() => {
  axios.get(url)
    .then(res => {
      setData(res.data);
      setLoading(false);
    })
    .catch(err => {
      setError(err.message);
      setLoading(false);
    });
}, []);

Best Practices

  • Use Axios for cleaner code
  • Handle loading and error states
  • Keep API logic in separate service files

Summary

  • API integration connects frontend & backend
  • Fetch and Axios are used for HTTP requests
  • Axios simplifies request handling
