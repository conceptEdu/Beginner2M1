Chapter 14: Introduction to Node.js

What is Node.js?

Node.js is a JavaScript runtime environment that allows JavaScript to run outside the browser. It is mainly used to build fast and scalable server-side applications.

Why Node.js?

  • Uses JavaScript on server-side
  • Fast execution using V8 engine
  • Event-driven and non-blocking I/O
  • Highly scalable

Features of Node.js

  • Asynchronous: Handles multiple requests without blocking
  • Single-threaded: Uses event loop for concurrency
  • Fast: Built on Google Chrome’s V8 engine
  • Open Source: Large community support

Node.js Architecture

Node.js works on an event-driven architecture. It uses an event loop to handle multiple client requests efficiently with a single thread.

Where Node.js is Used?

  • Web servers and APIs
  • Real-time applications (chat apps)
  • Streaming applications
  • Microservices

Installing Node.js

Node.js can be downloaded from the official website. After installation, it provides two main tools:

  • node – to run JavaScript files
  • npm – Node Package Manager

Check Installation

node -v
npm -v

Your First Node.js Program

// hello.js
console.log("Hello from Node.js");

Run the program using:

node hello.js

Advantages of Node.js

  • High performance
  • Same language for frontend and backend
  • Large ecosystem of libraries

Summary

  • Node.js allows JavaScript to run on server
  • It is fast, scalable, and asynchronous
  • Used widely in modern web applications
