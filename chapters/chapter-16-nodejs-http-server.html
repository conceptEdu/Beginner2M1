Chapter 16: Creating Server with Node.js (HTTP Module)

What is a Server?

A server is a program that listens for client requests and sends responses. In Node.js, we can create a server using the built-in http module.

HTTP Module

The http module allows Node.js to transfer data using the Hyper Text Transfer Protocol (HTTP). It is used to create web servers.

Creating a Basic HTTP Server

Example:

const http = require("http");

const server = http.createServer((req, res) => {
  res.write("Hello World from Node.js Server");
  res.end();
});

server.listen(3000, () => {
  console.log("Server running on port 3000");
});

After running this file using node filename.js, open browser and visit http://localhost:3000.

Request (req) Object

The req object contains information about the client request.

  • req.url – request URL
  • req.method – HTTP method (GET, POST)

Example:

console.log(req.url);
console.log(req.method);

Response (res) Object

The res object is used to send response to the client.

  • res.write() – send response data
  • res.end() – end the response

Setting Response Headers

res.writeHead(200, { "Content-Type": "text/html" });
res.write("<h1>Welcome to Node Server</h1>");
res.end();

Handling Different Routes

const http = require("http");

http.createServer((req, res) => {
  if (req.url === "/") {
    res.write("Home Page");
  } else if (req.url === "/about") {
    res.write("About Page");
  } else {
    res.write("404 Page Not Found");
  }
  res.end();
}).listen(3000);

Advantages of Node.js HTTP Server

  • Fast and lightweight
  • Handles multiple requests efficiently
  • Uses non-blocking I/O

Summary

  • HTTP module is used to create servers
  • createServer() creates a web server
  • req and res handle requests and responses
