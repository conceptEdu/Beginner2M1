Chapter 10: JavaScript String & Math Methods

String in JavaScript

A string is a sequence of characters used to store text. Strings can be written using single quotes, double quotes, or backticks.

Example:

let str1 = "JavaScript";
let str2 = 'MERN Stack';
console.log(str1, str2);

Common String Methods

1. length

Returns the length of the string.

let text = "Hello";
console.log(text.length); // 5

2. toUpperCase()

Converts string to uppercase.

console.log(text.toUpperCase()); // HELLO

3. toLowerCase()

Converts string to lowercase.

console.log(text.toLowerCase()); // hello

4. indexOf()

Returns the position of a specified value.

console.log(text.indexOf("e")); // 1

5. slice()

Extracts a part of a string.

let word = "JavaScript";
console.log(word.slice(0, 4)); // Java

6. replace()

Replaces specified text with new text.

let msg = "I love Java";
console.log(msg.replace("Java", "JavaScript"));

Math Object in JavaScript

The Math object provides mathematical constants and functions.

Common Math Methods

1. Math.round()

Rounds a number to the nearest integer.

console.log(Math.round(4.6)); // 5

2. Math.floor()

Returns the largest integer less than or equal to a number.

console.log(Math.floor(4.9)); // 4

3. Math.ceil()

Returns the smallest integer greater than or equal to a number.

console.log(Math.ceil(4.1)); // 5

4. Math.random()

Returns a random number between 0 and 1.

console.log(Math.random());

5. Math.max() and Math.min()

Returns the maximum and minimum value.

console.log(Math.max(10, 5, 20)); // 20
console.log(Math.min(10, 5, 20)); // 5

Summary

  • Strings are used to store text data
  • String methods help manipulate text
  • Math object provides useful mathematical operations
