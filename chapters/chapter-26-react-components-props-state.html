Chapter 26: React Components, Props & State

React Components

Components are the building blocks of a React application. Each component represents a part of the user interface and can be reused multiple times.

Types of Components

  • Functional Components
  • Class Components

Functional Components

Functional components are simple JavaScript functions that return JSX. They are the most commonly used components in modern React.

Example:

function Welcome() {
  return 
Welcome to React
;
}

export default Welcome;

Class Components

Class components are ES6 classes that extend React.Component. They were used earlier but are now mostly replaced by functional components with hooks.

Example:

import React, { Component } from "react";

class Welcome extends Component {
  render() {
    return 
Welcome to React
;
  }
}

export default Welcome;

What are Props?

Props (properties) are used to pass data from a parent component to a child component. Props are read-only.

Passing Props

// Parent Component
function App() {
  return ;
}

// Child Component
function Student(props) {
  return (
    

      Name: {props.name}, Course: {props.course}
    

  );
}

Props Destructuring

What is State?

State is a built-in React object used to store and manage component data that can change over time. When state changes, the component re-renders.

useState Hook

The useState hook is used to add state to functional components.

Syntax:

const [state, setState] = useState(initialValue);

useState Example

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <>
      
Count: {count}

      
    
  );
}

Difference Between Props and State

Props State
Passed from parent Managed inside component
Read-only Can be changed

Summary

  • Components build the UI
  • Props pass data to components
  • State manages dynamic data
  • useState is used in functional components
