Chapter 15: Node.js Modules & NPM

What are Modules?

Modules are reusable blocks of code that can be imported and used in different files. Node.js follows the CommonJS module system.

Types of Modules

  • Built-in Modules
  • User-defined Modules
  • Third-party Modules

1. Built-in Modules

Node.js provides several built-in modules to perform common tasks.

Examples:

  • fs – File System
  • http – Create web server
  • path – File path handling
  • os – Operating system info

Using require()

const fs = require("fs");

Creating User-defined Module

math.js

function add(a, b) {
  return a + b;
}

module.exports = add;

app.js

const add = require("./math");
console.log(add(5, 3));

Exporting Multiple Functions

module.exports = {
  add,
  subtract
};

What is NPM?

NPM (Node Package Manager) is used to install, manage, and share Node.js packages.

Initializing a Project

npm init

Installing Packages

Local Installation

npm install express

Global Installation

npm install -g nodemon

package.json

The package.json file stores project information and dependencies.

Using nodemon

nodemon app.js

Summary

  • Modules make code reusable
  • require() imports modules
  • NPM manages third-party packages
