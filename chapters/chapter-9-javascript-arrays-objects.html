Chapter 9: JavaScript Arrays & Objects

Arrays in JavaScript

An array is a special variable that can store multiple values in a single variable. Arrays are ordered and index-based.

Creating an Array

let subjects = ["HTML", "CSS", "JavaScript"];
console.log(subjects);

Accessing Array Elements

console.log(subjects[0]); // HTML
console.log(subjects[2]); // JavaScript

Updating Array Elements

subjects[1] = "Bootstrap";
console.log(subjects);

Common Array Methods

1. push()

Adds an element at the end of an array.

subjects.push("React");

2. pop()

Removes the last element from an array.

subjects.pop();

3. length

Returns the number of elements in an array.

console.log(subjects.length);

4. indexOf()

Returns the index of an element.

console.log(subjects.indexOf("JavaScript"));

Objects in JavaScript

Objects store data in key : value pairs. They represent real-world entities.

Creating an Object

let student = {
  name: "Rahul",
  age: 21,
  course: "MERN Stack"
};

Accessing Object Properties

console.log(student.name);
console.log(student["age"]);

Updating Object Properties

student.age = 22;
student.city = "Delhi";
console.log(student);

Array of Objects

JavaScript allows storing multiple objects inside an array. 

let students = [
  { name: "Amit", age: 20 },
  { name: "Neha", age: 22 }
];

console.log(students[0].name);

Difference Between Array and Object

Array Object
Stores multiple values Stores key-value pairs
Index-based access Key-based access
Ordered collection Unordered collection

Summary

  • Arrays store multiple values in order
  • Objects store data in key-value pairs
  • Arrays and objects are widely used in MERN applications
