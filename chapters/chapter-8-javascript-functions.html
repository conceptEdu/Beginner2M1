Chapter 8: JavaScript Functions

What is a Function?

A function is a reusable block of code designed to perform a specific task. Functions help in code reusability, modularity, and readability.

Function Syntax

function functionName(parameters) {
  // code to execute
}

Example:

function greet() {
  console.log("Welcome to JavaScript");
}
greet();

Function with Parameters

Parameters are used to pass values to a function.

Example:

function add(a, b) {
  console.log(a + b);
}
add(5, 3);   // Output: 8

Function with Return Statement

The return statement sends a value back to the function caller and stops function execution.

Example:

function multiply(a, b) {
  return a * b;
}

let result = multiply(4, 5);
console.log(result);   // Output: 20

Types of Functions

1. Function Declaration

function square(num) {
  return num * num;
}

2. Function Expression

let square = function(num) {
  return num * num;
};

Arrow Functions

Arrow functions provide a shorter syntax for writing functions.

Example:

let add = (a, b) => a + b;
console.log(add(10, 5));  // Output: 15

Benefits of Functions

  • Code reusability
  • Easy debugging
  • Better program structure

Summary

  • Functions are reusable blocks of code
  • They can accept parameters and return values
  • Arrow functions provide shorter syntax
