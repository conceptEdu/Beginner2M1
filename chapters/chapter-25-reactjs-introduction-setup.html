Chapter 25: React.js Introduction & Setup

What is React.js?

React.js is a JavaScript library developed by Facebook for building fast and interactive user interfaces. It is component-based and widely used for single-page applications (SPA).

Why Use React?

  • Component-based architecture
  • Fast rendering using Virtual DOM
  • Reusable UI components
  • Strong community support

Features of React.js

  • Declarative UI
  • Component-based structure
  • Virtual DOM
  • One-way data binding

What is Virtual DOM?

Virtual DOM is a lightweight copy of the real DOM. React updates the Virtual DOM first and then efficiently updates only the changed parts in the real DOM.

What are Components?

Components are independent and reusable pieces of UI. Each component returns JSX which describes what should appear on the screen.

Types of Components

  • Functional Components
  • Class Components

JSX (JavaScript XML)

JSX allows writing HTML inside JavaScript. It makes code easier to read and write.

Setting Up React Environment

Using Create React App

npx create-react-app myApp
cd myApp
npm start

This command creates a new React project with all required configurations.

Project Folder Structure

  • src/ – Application source code
  • public/ – Static files
  • package.json – Project dependencies
