Chapter 20: MongoDB Introduction & NoSQL Concepts

What is a Database?

A database is an organized collection of data that allows easy storage, retrieval, and management of information.

Types of Databases

  • SQL (Relational) Databases
  • NoSQL (Non-Relational) Databases

What is NoSQL?

NoSQL databases store data in a flexible, schema-less format. They are designed to handle large volumes of unstructured and semi-structured data.

Features of NoSQL

  • Schema-less data structure
  • High scalability
  • Fast performance
  • Distributed architecture

What is MongoDB?

MongoDB is a popular NoSQL database that stores data in JSON-like documents called BSON. It is widely used in MERN Stack applications.

MongoDB Key Terms

  • Database: Container for collections
  • Collection: Group of documents
  • Document: Data stored in key-value format

SQL vs MongoDB

SQL MongoDB
Tables Collections
Rows Documents
Fixed schema Schema-less

Installing MongoDB

MongoDB can be installed locally or used through MongoDB Atlas (cloud-based service).

Basic MongoDB Commands

Show Databases

show dbs

Create / Switch Database

use myDatabase

Show Collections

show collections

Summary

  • MongoDB is a NoSQL database
  • Stores data in documents
  • Highly scalable and flexible
