Chapter 11: JavaScript Date & DOM

JavaScript Date Object

JavaScript provides the Date object to work with dates and time. It allows us to get current date, time, and perform date-related operations.

Creating Date Object

let today = new Date();
console.log(today);

Common Date Methods

getFullYear()

let d = new Date();
console.log(d.getFullYear());

getMonth()

Returns month (0–11)

console.log(d.getMonth());

getDate()

console.log(d.getDate());

getDay()

Returns day of week (0–6)

console.log(d.getDay());

getHours(), getMinutes(), getSeconds()

console.log(d.getHours());
console.log(d.getMinutes());
console.log(d.getSeconds());

Setting Date Values

let newDate = new Date();
newDate.setFullYear(2026);
newDate.setMonth(0);
newDate.setDate(15);
console.log(newDate);

Introduction to DOM

DOM stands for Document Object Model. It represents the structure of an HTML document as a tree. JavaScript uses DOM to access and modify HTML elements dynamically.

Why DOM is Important?

  • Change HTML content
  • Change CSS styles
  • Handle events
  • Create dynamic webpages

Accessing HTML Elements

1. getElementById()

document.getElementById("demo");

2. getElementsByClassName()

document.getElementsByClassName("box");

3. getElementsByTagName()

document.getElementsByTagName("p");

4. querySelector()

document.querySelector(".container");

Changing HTML Content

document.getElementById("demo").innerHTML = "Hello DOM";

Summary

  • Date object is used to handle date and time
  • DOM allows JavaScript to interact with HTML
  • DOM makes webpages dynamic and interactive
